# 提案資料作成AIサポーター 仕様書

## 1. アプリ概要
アプリ名：提案サポーターAI「Propo-Draft（仮）」
目的：営業・企画担当者の提案書作成における時間的コストを削減し、属人化を防ぎ、社内標準テンプレートに準拠した高品質な提案書を最速で作成すること。

## 2. ターゲットユーザー
- 提案書のゼロイチ作成に時間がかかっている営業担当者
- 社内テンプレートの最新フォーマットやルールを把握しきれていないメンバー

## 3. コア機能（実装要件）
1. 与件整理機能: 乱雑なヒアリングメモから「背景」「課題」「要望」を構造化する。
2. 提案骨子作成機能: 社内テンプレートの「ご提案ポイント」「開発方針」のドラフトを生成する。
3. フォーマット・論理チェック機能: 敬称（貴社）、語尾、日付、論理破綻がないかを自動校閲する。

## 4. 前提となる社内テンプレート構成
1. ご挨拶
2. 与件整理（背景、ご依頼内容、現状分析、本質的な課題）
3. ご提案ポイント（強みや解決策を3〜4点で提示）
4. 開発方針（開発形態、推進プロセス）
5. トレードオフスライダー（優先順位の可視化）
